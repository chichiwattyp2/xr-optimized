How It Works

Get started with XR Home Chat in minutes

1

Clone & Deploy

Clone the repository and deploy to Vercel with one click. Set up your OpenAI API keys.

2

Enter VR Space

Open the app in your browser or VR headset. The 3D environment loads instantly.

3

Start Chatting

Use voice or text to interact with AI. Responses appear as floating UI elements in 3D space.

4

Customize

Modify the environment, add features, and create your own XR AI experiences.
